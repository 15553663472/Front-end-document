# 示例

## 手机号码

```
^1[\d]{10}$
```

## 一个合理的用户名正则表达式

用户名可以包含以下几种字符：

-   26 个大小写英文字母表示为 a-zA-Z
-   数字表示为 0-9
-   下划线表示为 \_
-   中划线表示为 -

根据以上条件得出用户名的表达式可以为：

```
[a-zA-Z0-9_-]+
```

## 邮箱地址

```
^(([^<>()[\]\\.,;:\s@\"]+(\.[^<>()[\]\\.,;:\s@\"]+)*)|(\".+\"))@((\[[0-9]{1,3}\.[0-9]{1,3}\.[0-9]{1,3}\.[0-9]{1,3}\])|(([a-zA-Z\-0-9]+\.)+[a-zA-Z]{2,}))$
```

## 匹配 HTML 标签及内容

```
<([a-zA-Z]+).*?>([\s\S]*?)</\1?>
```

## 匹配一个 URL 解析为协议、域、端口及相对路径

```
(\w+):\/\/([^/:]+)(:\d*)?([^# ]*)
```

## 匹配空行

```
^\s*$
```

## 中文

```
[\u4E00-\u9FA5]
```

## 车牌号

```
^[京津沪渝冀豫云辽黑湘皖鲁新苏浙赣鄂桂甘晋蒙陕吉闽贵粤青藏川宁琼使领A-Z]{1}[A-Z]{1}[A-Z0-9]{4}[A-Z0-9挂学警港澳]{1}$
```

## 十六进制颜色

```
^#?([a-fA-F0-9]{6}|[a-fA-F0-9]{3})$
```
